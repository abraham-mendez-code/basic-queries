This repository contains SQL practice exercises completed using MySQL Workbench and the Northwind sample database. Each exercise focuses on fundamental SQL concepts such as selecting data, filtering with WHERE, sorting with ORDER BY, pattern matching with LIKE, and understanding table relationships.

All exercises are organized into individual .sql files named by question number for clarity (e.g., 02_product_list.sql).

## Topics Covered

Selecting and projecting data

Sorting results

Filtering with operators, ranges, and patterns

Exploring products, categories, suppliers, and employees

Understanding table relationships in a relational database

## Structure

Each .sql file corresponds to one question from Workbook 7 and contains the query solution.
